# [janmaarten.com](https://janmaarten.com)

Personal portfolio and blog, built with [Ghost](https://ghost.org/) and a customized [Kyoto](https://themex.studio/kyoto/) theme. Hosted on [Pikapods](https://www.pikapods.com/). Deployed via [GitHub Actions](https://github.com/marketplace/actions/deploy-ghost-theme).

Issues with the site or its accessibility can be submitted [here](https://github.com/janmaarten-a11y/janmaarten.com/issues/new/choose).
